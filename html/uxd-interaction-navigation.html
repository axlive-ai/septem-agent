Agent 진입·종료·전환·재개 흐름이에요. 어디서 오고 어디로 가는지 맥락을 이어서 사용자가 흐름을 잃지 않게 해요.

상품 화면에서 진입
상품 화면에서 이어왔어요. 한도부터 볼까요?
응, 한도 알려줘
예상 한도
전세보증금 3억원 기준 2억 4,000만원이에요.
상품 추천
상품 추천은 맞춤상품 Agent가 이어서 도와드릴게요.

상품 화면에서 맥락을 이어받아 진입(Entry)하고, 대화 중 사용자가 다른 주제를 말하면 적합한 Agent로 전환할 것을 먼저 안내한 뒤 자연스럽게 연결(Agent Switch)해요. 사용자는 처음부터 다시 설명할 필요가 없어요.

Entry 진입

버튼·플로팅·딥링크 등 진입 경로에 따라 첫 인사가 달라져요. 어디서 왔는지 맥락이 있으면 그 맥락을 이어받아 첫 화면에서 바로 안내해요.

상품 화면에서 진입
상품 화면에서 이어왔어요. 한도부터 볼까요?
"금리 인하" 알림에서 진입
신용대출 금리가 0.3%p 내렸어요. 재약정하면 얼마나 아낄 수 있는지 볼까요?
DO
  • 진입 경로에 담긴 맥락(상품·알림 내용)을 인사말에 반영해요
  • 이미 알고 있는 정보는 다시 묻지 않아요
DON’T
  • 모든 진입 경로에서 같은 인사말을 반복해요
  • 진입 맥락을 무시하고 처음부터 대화를 시작해요

Exit 종료

진행 상태를 보존하며 안전하게 닫아요. 저장되지 않은 입력이 있으면 나가기 전에 저장 여부를 팝업으로 먼저 확인해요.

입력값이 없을 때바로 닫혀요
입력값이 있을 때저장 여부를 먼저 확인해요
전세보증금 3억원 기준 예상 한도는 2억 4,000만원이에요.
나가면 입력한 내용이 사라져요
지금까지의 계산 내용을 저장할까요?
DO
  • 저장되지 않은 입력이 있을 때만 확인 팝업을 띄워요
  • 입력값이 없으면 바로 닫아 불필요한 클릭을 줄여요
DON’T
  • 입력값 유무와 상관없이 항상 확인 팝업을 띄워요
  • 저장 여부를 묻지 않고 조용히 데이터를 날려요

Agent Switch 전환

현재 맥락을 넘겨 적합한 Agent로 전환해요. 사용자가 다른 주제를 이야기하면 어떤 Agent로 넘어가는지 먼저 안내한 뒤, 맥락을 그대로 전달하며 자연스럽게 연결해요.

상품 추천
상품 추천은 맞춤상품 Agent가 이어서 도와드릴게요.
맞춤상품 Agent에게 연결하고 있어요잠시만 기다려 주세요
DO
  • 어떤 Agent로 전환되는지 먼저 안내한 뒤 연결해요
  • 지금까지의 대화 맥락을 그대로 이어받아 다시 설명하지 않게 해요
DON’T
  • 아무 설명 없이 화면을 바꿔 낯선 상태로 던져놔요
  • 전환 후 이전 대화를 잊은 것처럼 처음부터 다시 물어봐요

Handoff Fallback 전환 실패 시

전환 대상 Agent가 응답하지 않거나 지연될 때도 사용자를 막다른 상태에 두지 않아요. 실패 사실을 솔직히 알리고, 원래 Agent가 계속 도와줄 수 있다는 대안을 함께 제시해요.

상품 추천
상품 추천은 맞춤상품 Agent가 이어서 도와드릴게요.
!지금은 맞춤상품 Agent 연결이 어려워요. 제가 이어서 도와드릴게요.
계속 도와줘잠시 후 다시 시도

Resume 재개

앱을 다시 열면 이어할 대화가 있다는 걸 배너로 먼저 알리고, 이전 입력을 복원해 중단 지점부터 이어서 진행해요.

이어할 대화가 있어요한도 조회 · 3분 전
이어하기
지난번 한도 조회를 이어서 진행할까요?
이어하기새로 시작
DO
  • 다시 열었을 때 이어할 대화가 있다는 걸 배너로 먼저 알려요
  • 이어하기와 새로 시작하기를 모두 선택할 수 있게 해요
DON’T
  • 사용자 의사와 상관없이 이전 대화로 강제로 이어가요
  • 오래 지난 대화도 구분 없이 계속 이어하기를 권해요
