Color

NDS의 색상 시스템은 디자이너와 개발자가 통일된 색상 이름을 사용하도록 도와줘요. 이 시스템을 활용하면 NH AI Agent 전반에서 일관된 UI를 쉽게 구현할 수 있어요.

기본 사용법 Usage

NDS의 색상은 HEX를 직접 쓰지 않고 의미 토큰(시맨틱)으로 사용해요. 토큰은 color 객체에서 가져와 적용할 수 있어요.

// 디자인 토큰 가져오기 import { color } from '@nhonce/tokens';

가져온 토큰을 컴포넌트의 색상 속성에 그대로 사용해요.

<Text style={{ color: color.font1 }} /> <View style={{ backgroundColor: color.bgContainer }} />

CSS 환경에서는 동일한 토큰을 변수로 사용해요.

/* CSS 변수 */ color: var(--color-font-1); background: var(--color-bg-container);
