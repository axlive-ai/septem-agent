종료/이탈/예외 · 사용자 케이스 정의

전체 Agent 공통으로 발생하는 종료·이탈·예외 상황을 정의해요.

Case A · 정상 종료

정의
고객이 업무를 완료하고 명시적으로 종료를 요청해요.
인식 신호
업무 완료 직후 "그만할래" 등 종료 의사 표현
웰컴 화면 방향
"도움이 됐으면 좋겠어요" 등 마무리 발화 후 대화 종료해요.

Case B · 비정상 이탈

정의
앱 종료·화면 전환으로 대화가 갑자기 중단돼요.
인식 신호
세션 미종료 상태에서 앱 백그라운드 전환 감지
웰컴 화면 방향
세션 상태를 자동 저장하고, 재진입 시 이어하기로 연결해요.

Case C · 시스템 오류

정의
API 응답 실패·네트워크 오류로 진행이 막혀요.
인식 신호
응답 타임아웃 · 서버 오류 응답
웰컴 화면 방향
오류를 안내하고 재시도를 유도하며, 반복 시 고객센터로 연결해요.

Case D · 세션 만료

정의
장시간 미사용으로 로그인 세션이 만료돼요.
인식 신호
세션 타임아웃 발생
웰컴 화면 방향
재로그인을 유도하고, 성공 시 이전 상태를 복원해요.
