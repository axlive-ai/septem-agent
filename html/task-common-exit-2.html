종료/이탈/예외 · 업무 설계서

전체 Agent 공통으로 수행하는 종료·이탈·예외 처리 업무를 정의해요.

업무 정의

업무명
종료/이탈/예외 처리 — 대화 중단 상황을 감지하고 적절히 대응해요.
업무 유형
상황 감지형 — 4가지 상황(정상종료·이탈·오류·세션만료)을 감지해 분기
수행 주체
시스템(상황 감지) + Agent(대응 발화)
완료 조건
상황별 대응 완료 후 대화 종료 또는 정상 흐름 복귀

업무 흐름

대화 진행상황 발생
상황 판별시스템
대응 실행Agent
분기 처리
정상 종료 → 종료 발화 후 홈 복귀
비정상 이탈 → 세션 저장 후 종료
시스템 오류 → 재시도 또는 고객센터 연결
세션 만료 → 재로그인 후 상태 복원

업무 단계별 상세

업무 1상황 판별
업무 조건
사용자 발화·앱 이벤트·시스템 신호 감지
정상종료·이탈·오류·만료 4가지 분류
UI 표현
발화 기반: "종료", "그만", "끝" 감지
이벤트 기반: 앱 백그라운드 전환 감지
화면 — 종료 의도 감지
‹화면
1
도움이 되셨다니 기쁘네요! 또 필요하시면 언제든 불러주세요. 😊
2
처음으로앱 종료
메시지 입력
1
종료 발화감사 인사와 함께 자연스럽게 대화를 마무리해요.
2
후속 액션 칩처음으로 돌아가거나 앱을 종료할 수 있어요.
예외 처리
신호 불명확 시 "계속하시겠어요?" 재확인
2회 무응답 시 자동 이탈 처리
발생 조건
"다른 건 없나요?"처럼 종료 의도가 불명확한 발화
화면 — 종료 의도 불명확 재확인
‹화면
1
혹시 지금 대화를 마치실 건가요?
2
네, 종료할게요아니요, 더 도와주세요
메시지 입력
1
재확인 발화종료 의도가 불명확하면 먼저 확인해요.
2
명확한 선택지종료 여부를 칩으로 명확히 선택할 수 있어요.
업무 2상황별 대응 실행
업무 조건
정상 종료: 마무리 발화 + 세션 저장
이탈: 24시간 이어하기 예약
오류: 재시도 유도
UI 표현
이탈 시: "이어하기" 알림 예약 안내
오류 3회 초과: 고객센터 연결 버튼
화면 — 이탈 시 이어하기 예약
‹화면
1
잠깐 나가시는군요. 진행 중이던 내용을 저장해뒀어요. 24시간 안에 돌아오시면 이어서 도와드릴게요!
· 전세대출 보증 추천 — 진행 중 저장됨
메시지 입력
1
세션 저장 알림이탈 시 진행 내용을 자동 저장하고 고객에게 안내해요.
2
이어하기 안내24시간 이내 재진입 시 이어하기가 가능해요.
예외 처리
세션 저장 실패 시 이어하기 불가 안내
API 오류 지속 시 고객센터 연결
발생 조건
네트워크 단절로 세션 저장 실패
화면 — 저장 실패 안내
‹화면
1
죄송해요, 일시적인 오류로 진행 내용을 저장하지 못했어요.
2
다시 시도고객센터 연결
메시지 입력
1
오류 안내저장 실패 시 고객에게 명확히 알려요.
2
대안 제공재시도 또는 고객센터 연결로 해결할 수 있어요.
