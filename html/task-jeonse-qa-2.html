전세 여정 전반의 실시간 질의응답과 Sub Agent 연결을 수행하는 업무를 정의해요.

업무 정의

업무명
질의응답 및 단계 연결 — RAG 기반 답변 제공 및 필요 시 Sub Agent 연결
업무 유형
질의응답형 — RAG 검색 + 조건부 라우팅
수행 주체
Agent(RAG 검색·답변 생성) + 고객(질문)
완료 조건
답변 제공 완료 또는 Sub Agent 연결 완료
