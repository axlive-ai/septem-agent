AI종합자산관리 Master

고객의 자산진단부터 포트폴리오 조정·절세 안내까지 통합 지원하는 Master Agent예요.

Agent 개요

목적
개인화되지 않은 비효율적인 금융정보 제공 구조의 한계를 극복하고, 포트폴리오 진단·설계 고도화 등 실질적인 자산관리 서비스를 제공해요.
추진 배경
대면 중심 자산관리를 대면+비대면 Omni 채널로 확장하고, Rule-Based 단순 수익률 계산에서 AI Base + Rule-Based 통합 서비스로 고도화할 필요가 있었어요.
대상 고객
WM 시스템 대상 고객 + 올원뱅크 마이데이터 동의 고객
화면 형태
올원뱅크 + WM 시스템 동시 탑재 — 채팅 UI + 자산 대시보드

역할 및 기능 범위

주요 기능
모닝브리핑 Sub Agent — 개인화 시황 브리핑 제공
포트폴리오 진단 Sub Agent — 자산·부채 통합 진단
포트폴리오 설계 Sub Agent — 맞춤형 포트폴리오 조정 제안
절세 Sub Agent — 절세 방향 및 과세이연 상품 솔루션 제안
서비스 흐름
대면→Omni — WM시스템 + NH올원뱅크 동시 탑재
Rule-Based→AI Base — 개인화·심층분석 필요 영역에 AI Modeling 적용
자산→자산+부채 — 부채까지 반영한 종합 자산관리 서비스
