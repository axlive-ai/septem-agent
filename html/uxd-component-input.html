선택형 질문은 Chip으로, 여러 값이 필요할 땐 Slot Input으로 하나씩 채워가요. 자유롭게 입력할 땐 Text Input이 Focus(Green 보더)로 활성화돼요. 상황에 맞는 입력 방식을 자연스럽게 오가요.

Text Input 텍스트 입력

자유 텍스트를 입력하는 필드예요. Default는 회색 보더(#DDDDDD), Focus 시 Green(#19973C), Error 시 Red(#E24949) 보더로 상태를 구분해요. radius는 12를 사용해요.

Voice Input 음성 입력

말로 입력하는 동안 인식 상태를 파형으로 보여줘요. Idle 상태는 마이크 아이콘만, Listening 상태는 Green 보더와 파형 애니메이션으로 듣고 있음을 알려요. 인식이 끝나면 텍스트로 변환해 Text Input과 동일하게 처리해요.

Slot Input 슬롯 입력

보증금·소득처럼 여러 값을 항목별로 채우는 입력이에요. 채워진 슬롯은 진한 텍스트(font-1)로, 미입력 슬롯은 옅은 안내 문구(font-5)로 구분해요. 슬롯은 한 번에 하나씩, 순서대로 요청해요.

Chip 칩

예측 가능한 보기를 탭 한 번으로 선택하는 칩이에요. 선택된 칩은 Active 스타일(#111 배경·흰 텍스트)로 강조하고, 보기는 3~6개로 제한해 한눈에 비교할 수 있게 해요.
