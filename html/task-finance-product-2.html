맞춤상품안내 · 업무 설계서

ML 추천 결과를 활용해 니즈 수집부터 가입까지 지원하는 업무를 정의해요.

업무 정의

업무명
상품 안내 — 니즈를 수집하고 맞춤 상품을 추천·비교·가입까지 연결해요.
업무 유형
추천형 (슬롯 필링 + 검색·랭킹)
수행 주체
Agent(니즈 수집·상품 검색·추천) + 고객(조건 입력·선택)
완료 조건
고객이 추천 상품을 확인하고 가입·비교·종료 중 하나를 선택

업무 흐름

진입fo-start
니즈 수집3슬롯
추천·가입자동
분기 처리
가입 선택 → 상품 가입 프로세스 이동
비교 요청 → 비교표 표시
결과 없음 → 조건 완화 제안

업무 단계별 상세

업무 1니즈 슬롯 수집 (목표기간·예치금액·우대조건)
업무 조건
슬롯 순서대로 수집, 건너뛰기 불가
유효값 입력 즉시 다음 슬롯 자동 진행
허용 범위: 목표기간 1~120개월 · 예치금액 1만원 이상
UI 표현
슬롯 진행 카드 — 완료·활성·대기 3단계
슬롯별 빠른 선택 Chip 제공
예외 처리
범위 외 입력 → 허용 범위 안내 후 재수집
업무 2상품 검색 및 추천
업무 조건
슬롯 3개 완료 즉시 자동 검색
조건 매칭 상품 최대 5개 추출, 1순위 강조
UI 표현
1순위 추천 카드 — 금리 강조(빨강 Bold)
2순위 이하 카드 — 축소 표시
예외 처리
검색 결과 없음 → 조건 완화 제안 + 조정 Chip
업무 3비교 및 가입 연결
업무 조건
고객 요청 시 2개 이상 상품 비교표 생성
가입 선택 시 즉시 가입 프로세스 연결
UI 표현
상품 비교표 — 금리 행 하이라이트
가입하기 버튼 — Primary
예외 처리
투자 상품의 경우 투자 권유 규정 준수 — 확정 수익률 표현 금지
