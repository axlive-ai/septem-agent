웰컴대화 · 업무 설계서

전체 Agent 공통으로 수행하는 웰컴 업무를 정의해요.

업무 정의

업무명
웰컴 대화 — 고객 상태를 판별하고 적절한 진입 발화를 제공해요.
업무 유형
조건 분기형 — 고객 상태(이력·세션)에 따라 4가지 케이스로 분기
수행 주체
Agent(상태 판별·발화 생성) + 시스템(이력·세션 조회)
완료 조건
웰컴 발화 표시 후 고객이 다음 액션을 선택

업무 흐름

앱 진입
상태 판별시스템
웰컴 발화Agent
분기 처리
첫 방문 → 온보딩 포함 웰컴
진행 중 세션 존재 → 이어하기 카드
최근 이용 이력 존재 → 개인화 인사
알림 유입 → 맥락 직행

업무 단계별 상세

업무 1고객 상태 판별
업무 조건
진입 즉시 실행 — 세션·이력 데이터 기반 분류
분류 결과에 따라 업무 2 분기
UI 표현
고객 상태 4가지(첫방문·이탈·재방문·알림) 자동 판별
분류 로직은 백그라운드 실행 — 고객에게 노출 안 됨
화면 — 고객 상태 판별 중
‹화면
잠깐만요, 맞춤 인사를 준비할게요!
· 이전 이용 내역 확인 중...
메시지 입력
1
자동 판별고객이 어떤 상태로 진입했는지 백그라운드에서 즉시 분류해요.
2
Fallback 처리데이터가 없으면 첫방문으로 처리해 온보딩을 시작해요.
예외 처리
세션 데이터 누락 시 첫방문으로 fallback
이력 불일치 시 재방문 우선 처리
발생 조건
마이데이터 미연동 등 데이터 없는 경우
화면 — 데이터 없을 때 Fallback
‹화면
1
안녕하세요! NH AI Agent에 오신 걸 환영해요.
2
대출 상담상품 추천자산 관리
메시지 입력
1
Fallback 발화이력이 없으면 첫방문 온보딩 발화로 자동 전환해요.
2
서비스 선택 칩어떤 도움이 필요한지 바로 선택할 수 있어요.
업무 2웰컴 발화 생성 및 표시
업무 조건
판별 결과에 따라 4가지 발화 중 하나 선택
재방문·알림 진입 시 이전 맥락 참조
UI 표현
첫방문: 서비스 소개 + 선택 칩
재방문: "지난번 이어서" 참조 발화
알림 진입: 알림 내용 즉시 반영
화면 — 재방문 고객 웰컴
‹화면
1
다시 오셨군요! 지난번에 전세대출 상담을 하셨는데, 이어서 도와드릴까요?
2
네, 이어서요아니요, 다른 걸로
메시지 입력
1
맥락 참조 발화이전 이용 내용을 참조해 개인화된 인사말을 생성해요.
2
빠른 선택 칩이어하기 여부를 칩으로 바로 선택할 수 있어요.
예외 처리
생성 실패 시 기본 발화로 대체
맥락 불일치 시 첫방문 발화 fallback
발생 조건
이전 이용 내역은 있지만 맥락 매칭 실패
화면 — Fallback 기본 발화
‹화면
1
안녕하세요! 오늘은 어떤 도움이 필요하세요?
대출 상담상품 추천자산 관리소비자보호
메시지 입력
1
Fallback 발화맥락 매칭이 실패하면 기본 발화로 대체해요.
2
전체 서비스 칩모든 서비스를 선택할 수 있는 칩을 제공해요.
