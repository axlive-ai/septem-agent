Typography

NDS의 타이포그래피는 Pretendard를 기본 서체로, 모바일에서 읽기 쉽고 일관된 텍스트 경험을 제공해요.

기본 사용법

NDS의 모든 텍스트는 Pretendard를 기본 서체로 사용해요. 국문·영문·숫자·특수문자를 포함하고, 경량 폰트로 로딩 속도와 가독성을 함께 챙겼어요. 텍스트 스타일은 typography 토큰에서 가져와 일관되게 적용할 수 있어요.

// 타이포그래피 토큰 가져오기 import { typography } from '@nhonce/tokens';

가져온 스타일을 컴포넌트에 그대로 적용해요.

<Text style={{ ...typography.h1 }}>NH AI Agent UX 디자인시스템</Text>
Pretendard 프리텐다드 0123!@#$
SemiBoldMediumRegular
