사후관리 · 사용자 케이스 정의

계약 이후 행정 절차 안내 Sub Agent가 마주하는 사용자 케이스를 정의해요.

Case A · 계약 직후 진입

정의
전세 계약을 막 체결한 고객이에요.
인식 신호
깡통전세방지 단계 완료 · 계약 완료 표시
웰컴 화면 방향
입주일·확정일자 신청 여부부터 친절하게 안내해요.

Case B · 입주 전 알림 확인

정의
입주 예정일이 다가와 알림으로 진입한 고객이에요.
인식 신호
D-day 알림 클릭 진입
웰컴 화면 방향
임박한 절차를 강조해 "오늘 이사 끝나셨나요?" 식으로 적시 제안해요.

Case C · 등기 변동 알림 확인

정의
등기부 변동 알림으로 진입한 고객이에요.
인식 신호
등기 변동 이벤트 발생 후 알림 클릭
웰컴 화면 방향
변동 내용을 즉시 보여주고 필요한 조치를 안내해요.

Case D · 보증보험 리마인드

정의
보증보험 가입을 잊고 있는 고객이에요.
인식 신호
확정일자 완료 후 보증보험 미가입 상태 장기 지속
웰컴 화면 방향
리마인드 메시지로 가입을 다시 권장해요.
