종합자산관리 AI가 마주하는 사용자 케이스를 정의해요.

Case A · 첫 자산 진단

정의
처음 종합자산관리를 이용하는 고객이에요.
인식 신호
마이데이터 미연동 · 이용 이력 없음
웰컴 화면 방향
마이데이터 연동 이점을 설명하고 동의를 유도해요.

Case B · 모닝브리핑 미확인

정의
매일 브리핑을 받지만 확인하지 않는 고객이에요.
인식 신호
최근 7일 브리핑 미열람
웰컴 화면 방향
누적 변동사항을 요약해 핵심만 보여줘요.

Case C · 정기 점검 재방문

정의
주기적으로 자산을 점검하는 고객이에요.
인식 신호
월 1회 이상 정기 방문 패턴
웰컴 화면 방향
이전 진단·설계 이력을 참조해 변화 추이를 보여줘요.

Case D · 알림 통한 진입

정의
리밸런싱 알림이나 세금 알림으로 진입한 고객이에요.
인식 신호
알림 클릭 진입 · 특정 Sub Agent 맥락 존재
웰컴 화면 방향
알림 내용을 그대로 이어받아 해당 Sub Agent로 직행해요.
