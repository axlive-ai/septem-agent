AI소비자보호 Master

금융사기 예방부터 위협 탐지·사후 분석까지 全 과정을 통합 지원하는 Master Agent예요.

Agent 개요

목적
AI 기술을 활용한 예방 패러다임 전환(수동→능동적 예방)을 이루고, 무과실 배상 책임 제도 도입(예정)에 따른 강화된 사회적 책임을 이행해요.
추진 배경
공통 예방정보 안내 메뉴를 운영 중이나 실시간 탐지·신규 사기 패턴 대응 체계가 미흡했고, 고객군별 취약 사기 유형 맞춤 대응이 필요했어요.
대상 고객
올원뱅크 전 고객 — 특히 사기 취약 고객군(고령층·지방 거주자 등)
화면 형태
올원뱅크 인증/보안 메뉴 + 실시간 팝업 경고

역할 및 기능 범위

주요 기능
금융사기 예방 Sub Agent — 고객 프로파일링 기반 맞춤형 예방 정보 제공
피싱 탐지 Sub Agent — 실시간 스팸·악성 URL 탐지 및 팝업 경고
금융사기 시나리오 추출 Sub Agent — 신종 사기 패턴 발굴·FDS 반영
서비스 흐름
사전예방(정보안내) — 고객군별 맞춤형 금융사기 예방 안내
사전예방(실시간탐지) — 피싱·악성 URL 실시간 식별 및 전면 팝업 경고
사후분석(탐지고도화) — 생성형 AI 기반 신종 사기 패턴 발굴
