포트폴리오 진단 · 사용자 케이스 정의

당·타행 자산과 부채를 통합 진단하는 Sub Agent가 마주하는 사용자 케이스를 정의해요.

Case A · 첫 통합 진단

정의
처음 자산·부채 통합 진단을 받는 고객이에요.
인식 신호
마이데이터 연동 직후 · 진단 이력 없음
웰컴 화면 방향
통합 분석 결과를 처음 접하므로 항목별로 차근차근 설명해요.

Case B · 부채 데이터 누락

정의
부채 데이터가 일부 연계되지 않은 고객이에요.
인식 신호
신규 부채 연계 데이터 부재
웰컴 화면 방향
현재 가능한 범위 내에서 진단하고 추가 연계를 권장해요.

Case C · 정기 비교 진단

정의
이전 진단과 비교하려는 고객이에요.
인식 신호
이전 진단 이력 존재
웰컴 화면 방향
이전 결과와 현재를 나란히 비교해 변화를 보여줘요.

Case D · 브리핑 연계 진단

정의
모닝브리핑에서 바로 연결된 고객이에요.
인식 신호
브리핑에서 "포트폴리오 보기" 선택
웰컴 화면 방향
브리핑에서 언급된 이슈를 진단 결과와 연결해 설명해요.
