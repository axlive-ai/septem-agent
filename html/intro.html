Introduction

이 사이트는 NH농협은행에 적용할 "AI 에이전트"를 어떻게 만들지 정리해놓은 자료예요. AI 에이전트가 뭔지 모르셔도 괜찮아요. 천천히 설명해드릴게요.

AI 에이전트가 뭐예요?

쉽게 말하면
은행 직원처럼 고객과 대화하면서 일을 처리해주는 AI예요. 예를 들어 "대출 이자 얼마나 나와?"라고 물으면 필요한 정보를 차례로 물어보고, 계산해서 답을 알려줘요. 사람이 창구에서 상담받는 것과 비슷한데, 그걸 앱 안에서 채팅으로 하는 거예요.
왜 필요해요?
지금까지는 고객이 메뉴를 직접 찾아 들어가서 정보를 입력해야 했어요. AI 에이전트를 쓰면 고객이 그냥 말하듯 물어보기만 하면 AI가 알아서 안내해줘요.
이 사업은 뭐예요?
NH농협은행이 올원뱅크 앱에 이런 AI 에이전트를 여러 개 만들어 넣는 사업이에요. 2026년 8월에 본격적으로 시작되고, 셉템은 그 전에(7월 한 달) 미리 화면과 대화 흐름을 설계해두는 일을 맡았어요.

이 사이트에서 무엇을 볼 수 있나요?

왼쪽 메뉴를 보시면 5개의 큰 카테고리가 있어요. 순서대로 보시면 하나의 AI 에이전트가 "왜 필요한지"부터 "실제로 어떤 화면으로 나오는지"까지 쭉 이어서 이해할 수 있어요.

왼쪽 메뉴 — 5단계로 구성돼 있어요
① Agent Profile (소개) — 이 AI가 누구인지, 무슨 일을 하는지 한 페이지로 요약해놓은 자료예요. 명함 같은 거예요.
② Agent Task Design (설계) — 이 AI가 실제로 어떤 순서로 고객과 대화하고, 어떤 상황에서 어떻게 반응하는지 자세히 적어놓은 자료예요.
③ Agent UX Design (디자인 기준) — 화면에 쓰일 색깔, 글씨체, 버튼 모양 같은 디자인 규칙이에요. (이 부분은 디자인팀이 별도로 작업해요)
④ Agent Wireframe (화면 그림) — 실제 휴대폰 화면이 어떻게 생겼는지 그려놓은 자료예요.
⑤ Agent Prototype (체험판) — 나중에는 실제로 눌러보면서 체험할 수 있는 화면도 추가될 거예요.
어떤 AI 에이전트들이 있나요?
금융비서 — 예금·적금·대출 같은 상품을 추천해주고, 대출 이자 같은 걸 계산해줘요.
전세대출 — 전세 계약을 준비할 때 대출 한도부터 전세 사기 위험까지 도와줘요.
소비자보호 — 보이스피싱이나 사기를 당했을 때 빠르게 도와주고, 미리 예방 정보도 알려줘요.
종합자산관리 — 내 자산을 한눈에 보여주고, 더 나은 방향으로 조정하는 걸 도와줘요.
