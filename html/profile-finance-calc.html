AI금융계산기

대출이자·DSR·중도상환수수료 등 복합 금융계산을 대화로 수행하고 결과를 쉽게 설명해주는 Sub Agent예요.

Agent 개요

목적
고객이 직접 이해하고 계산해야 했던 대출이자·DSR·중도상환수수료를 대화 방식으로 손쉽고 빠르게 처리해요.
추진 배경
대출 조건·상환 비용 등 금융용어나 계산에 필요한 항목들을 고객이 직접 이해하고 계산해야 하는 어려움이 있었어요.
대상 고객
대출 조건·상환 비용 계산이 필요한 고객
화면 형태
인앱 새 창 — 채팅 UI + 결과 카드

역할 및 기능 범위

주요 기능
대출이자 계산 (원리금균등·원금균등·만기일시)
DSR(총부채원리금상환비율) 계산
중도상환수수료 계산 (기존 여신계좌 연계)
계산 결과 인라인 설명 및 용어 안내
서비스 흐름
진입 — 올원뱅크 내 AI 금융 계산기 화면 진입
요청 — 고객이 대화창에 자연어로 계산 요청 입력
추론 — AI가 의도·핵심 개체 분석, 슬롯 필링
완료 — 계산된 값을 대화창에 즉시 안내

제약 및 특이사항

제약 및 개발 협의 필요
계산 유형별 필수 입력항목 상이 — 유형마다 슬롯 구성 다름
중도상환수수료 계산은 기존 여신계좌 연계 필요 (계정계 API)
계산 결과는 참고용이며 실제 대출 조건과 상이할 수 있음 — 법적 고지 문구 필요
