피싱 탐지

스팸·악성 URL 등 피싱 위협을 탐지하고 경고하는 Sub Agent예요.

Agent 개요

목적
올원뱅크 앱에 탑재된 SDK 기반 피싱 탐지 시스템을 연계하여 악성 앱 실행 및 스미싱 문자 인입 시 위험을 즉시 탐지하고 전면 팝업으로 경고해요.
추진 배경
실시간 탐지 기능이 미공 상태였고, 정보안내 메뉴를 통한 외부 유료 서비스 연계 수준으로만 운영되고 있었어요.
대상 고객
올원뱅크 앱 사용 고객 전체 — 스미싱·악성 앱 위협에 노출된 모든 고객
화면 형태
실시간 전면 팝업 경고 + 올원뱅크 UI/UX 가이드라인 적용 알림 화면

역할 및 기능 범위

주요 기능
솔루션 SDK가 실시간 모니터링 중 위험 감지 → 탐지 정보를 당행 앱 서버로 즉시 전송
당행 AI Agent가 경고 화면을 통해 능동적으로 안내 (전면 팝업·이체 기능 임시 차단)
탐지된 로그(내용·위험 URL·시간) 내부 모니터링 DB 적재
서비스 흐름
단말기 — SDK가 실시간 위험 식별
통신 — 탐지 정보를 당행 앱 서버로 즉시 전송
UX — 당행 앱 내 경고 팝업 노출 및 이체 기능 임시 차단
사후 — 탐지 로그 내부 모니터링 DB 적재
