대화형 질의응답

전세 여정 전반의 상담과 단계별 서비스 연결을 지원하는 Sub Agent예요.

Agent 개요

목적
전세 여정(준비~입주) 전반에 걸쳐 실시간 질의응답을 제공하고, 고객 발화 의도를 파악하여 필요한 단계의 서비스로 자연스럽게 연결해요.
추진 배경
고객이 전세 여정 중 언제든 질문하고 다음 단계로 이동할 수 있는 통합 상담 창구가 필요했어요.
대상 고객
전세 관련 궁금증이 있는 모든 단계의 고객
화면 형태
전세 플래너 내 대화형 채팅 UI

역할 및 기능 범위

주요 기능
전세 여정 전반 실시간 질의응답 제공
고객 발화 의도 파악 → 필요한 Sub Agent로 연결
전세 관련 용어·필요 서류·절차 RAG 기반 답변
단계별 추가 정보가 필요한 경우 관련 Agent 호출
서비스 흐름
고객이 전세 플래너 내 대화창에 질문 입력
AI가 질문 의도와 현재 전세 여정 단계 파악
필요한 정보는 바로 답변, 추가 지원 필요 시 관련 Agent 연결
고객이 연결된 서비스에서 바로 다음 단계 진행
