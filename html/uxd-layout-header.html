실제 화면에서 자주 쓰는 헤더 유형과 그 아래 콘텐츠가 시작되는 간격을 보여주는 예시예요. 기본 화면(Top-Basic)은 뒤로가기·타이틀·홈/메뉴로 구성하고 콘텐츠를 24px 아래에서 시작하며, 프로세스 화면(Top-Process)은 우측 ‘취소’로 흐름을 빠져나가요. 풀페이지 팝업(Top-Full popup)은 타이틀·X로 닫고 콘텐츠 집중을 위해 간격을 16px로 좁혀요.

Basic

일반 서비스 페이지에서 사용하는 기본 헤더예요. 타이틀은 16 / Semibold / -2% / #121212로 가운데 정렬해요.

Process

프로세스가 진행되는 페이지에서 사용하는 헤더예요. 우측 버튼은 14 / Regular / -2% / #343434로 표시해요.

Full-Popup

풀페이지 팝업에서 사용하는 헤더예요. 타이틀은 16 / Semibold / -2% / #121212로 왼쪽 정렬해요.
