AI전세대출 Master · 사용자 케이스 정의

전세대출 AI가 마주하는 사용자 케이스를 정의해요.

Case A · 전세 준비 시작

정의
전세 계약을 준비하기 시작한 고객이에요.
인식 신호
전세대출 AI 최초 진입 · 5단계 여정 이력 없음
웰컴 화면 방향
5단계 여정(준비~입주)을 소개하고 현재 위치를 안내해요.

Case B · 여정 중간 이탈

정의
여정의 특정 단계에서 이탈한 고객이에요.
인식 신호
특정 단계(예: 매물탐색) 완료 후 다음 단계 미진입
웰컴 화면 방향
직전 단계 결과를 요약해 보여주고 다음 단계를 제안해요.

Case C · 단계 완료 후 재방문

정의
이전 단계를 완료하고 다시 방문한 고객이에요.
인식 신호
보증 추천 등 이전 단계 완료 데이터 존재
웰컴 화면 방향
완료된 단계 결과를 참조해 다음 단계로 자연스럽게 연결해요.

Case D · 위험 알림 진입

정의
깡통전세 위험 알림이나 등기 변동 알림으로 진입한 고객이에요.
인식 신호
알림 클릭 진입 · 위험 등급 데이터 존재
웰컴 화면 방향
위험 상황을 즉시 안내하고 대응 단계로 직행해요.
