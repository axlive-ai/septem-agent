절세 · 사용자 케이스 정의

금융소득종합과세 절세 안내 Sub Agent가 마주하는 사용자 케이스를 정의해요.

Case A · 상한 도달 알림 첫 확인

정의
처음 금융소득종합과세 상한 알림을 받는 고객이에요.
인식 신호
금융소득종합과세 기준금액 근접 첫 감지
웰컴 화면 방향
기준금액과 현재 누적 비율을 차근차근 설명해요.

Case B · 소득 데이터 일부 누락

정의
배당·근로소득 등 일부 데이터가 없는 고객이에요.
인식 신호
내부 데이터 외 소득 정보 부재
웰컴 화면 방향
고객 직접 입력을 요청하거나 추정치로 안내해요.

Case C · 과세이연 상품 보유

정의
이미 ISA·IRP 등을 보유한 고객이에요.
인식 신호
기존 과세이연 상품 가입 데이터 존재
웰컴 화면 방향
추가 한도나 신규 상품을 우선 제안해요.

Case D · 포트폴리오 연계 절세

정의
리밸런싱과 함께 절세를 고려하는 고객이에요.
인식 신호
포트폴리오 설계 단계에서 연계 진입
웰컴 화면 방향
리밸런싱과 절세를 통합 실행하는 방안을 제안해요.
