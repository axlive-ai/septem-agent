대화형 질의응답 · 사용자 케이스 정의

전세 여정 전반의 질의응답 Sub Agent가 마주하는 사용자 케이스를 정의해요.

Case A · 단순 질문

정의
전세 용어나 절차에 대한 단순 질문이에요.
인식 신호
RAG로 즉시 답변 가능한 일반 질문
웰컴 화면 방향
RAG 기반으로 즉시 답변하고 추가 질문을 유도해요.

Case B · 복합 질문 중 이탈

정의
복잡한 질문에 답변 중 이탈한 고객이에요.
인식 신호
긴 답변 제공 중 화면 이탈
웰컴 화면 방향
이어하기 시 이전 답변 맥락을 유지한 채 대화를 재개해요.

Case C · 반복 방문 질문자

정의
여러 차례 질문해온 고객이에요.
인식 신호
동일 세션 내 3회 이상 질문
웰컴 화면 방향
누적된 맥락을 참조해 더 구체적인 답변을 제공해요.

Case D · 단계 전환 필요 질문

정의
답변만으로 해결 안 되고 Sub Agent 연결이 필요한 질문이에요.
인식 신호
특정 서비스 실행을 요구하는 질문 ("내 한도 알려줘" 등)
웰컴 화면 방향
답변과 함께 해당 Sub Agent로 자연스럽게 연결해요.
