Inline Confirm 인라인 확인

채팅 흐름 안에서 물어보는 확인 카드예요. Agent 말풍선과 같은 위치·radius(좌상단 4)를 써서 대화의 연장선처럼 보이게 하고, 화면을 가리지 않아요. 이체·신청처럼 되돌릴 수는 있지만 확인이 필요한 대부분의 실행에 이 방식을 기본으로 써요.

Dialog 전체 확인

화면 전체를 덮는 예외적인 확인이에요. 해지·한도 상향처럼 되돌릴 수 없거나 영향이 큰 실행에만 제한적으로 써요. 버튼은 채우지 않은 아웃라인 스타일을 쓰고, 취소(회색)를 왼쪽에, 실행(그린)을 오른쪽에 둬요. 대화형 경험의 기본값이 아니라 꼭 필요할 때만 사용해요.

해지 신청을 도와드릴게요.

Toast 토스트

짧은 비차단 완료 알림이에요. 검정 배경(#111)에 흰 텍스트로 2~3초간 노출 후 화면을 가리지 않고 자동으로 사라져요. 실행 취소처럼 되돌릴 수 있는 동작은 액션 링크를 함께 붙이고, 실패는 Red(--error)로 구분해요.
