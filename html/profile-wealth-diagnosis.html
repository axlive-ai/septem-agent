포트폴리오 진단

당·타행 자산과 부채를 통합한 종합 포트폴리오를 진단하는 Sub Agent예요.

Agent 개요

목적
당·타행 자산과 부채를 통합 분석하여 기대수익률·예상 부채 부담액·유사 고객군 비교 등 종합 진단을 제공해요.
추진 배경
현행 WM시스템 내 포트폴리오 진단은 보유자산 비율 중심 룰 기반으로 운영되었고, 부채 미반영·개인화 한계를 보완할 필요가 있었어요.
대상 고객
WM 시스템 대상 고객 — 종합 자산·부채 진단이 필요한 투자 고객
화면 형태
WM 시스템 포트폴리오 진단 화면 + 올원뱅크 Chat Loop

역할 및 기능 범위

주요 기능
당행 자산(WM 시스템) + 타행 자산·부채(마이데이터) 통합 분석
기대수익률 산출 — 자산군별 실제 수익률 또는 약정 수익률 기반
부채부담액 산출 — 실제 부담 중인 원금 상환액·이자액 기반
유사 고객군 비교 분석 및 종합 해석 생성 (LLM)
서비스 흐름
WM 시스템 적재 데이터 호출 (수신·펀드·방카 등)
신규 연계 데이터 — 고객별 부채 데이터 (관련 부서 협조)
타행 자산·부채 — 마이데이터 동의 고객 기준 당·타행 합산
통합 분석 결과 LLM 해석 생성 → 고객 전달

제약 및 특이사항

제약 및 개발 협의 필요

마이데이터 동의 고객 대상 — 미동의 고객 서비스 범위 별도 정의
