깡통전세방지

등기부·권리관계·전세가율 등을 분석해 전세 위험을 진단하는 Sub Agent예요.

Agent 개요

목적
등기부등본·권리관계·전세가율 분석을 통해 전세 계약 전 위험 요소를 파악하고 안전한 계약 판단을 지원해요.
추진 배경
전세 사기 급증으로 인한 고객 피해 예방이 필요했고, 등기부 분석·권리관계 확인이 전문 지식 없이는 어려운 상황이었어요.
대상 고객
전세 계약 전 위험 진단이 필요한 고객
화면 형태
기존 올원뱅크 제휴 서비스(전세 안전진단) 연계

역할 및 기능 범위

주요 기능
전세금·물건지 주소 등 기본 데이터 입력 수집
등기부등본 확인 후 근저당권·가압류 등 위험 요소 판독
해당 주택 시세(공시지가) 대비 전세금+융자금 비율 자동 계산 → 위험도 산출
서비스 흐름
깡통전세방지 메뉴로 진입 → 설명 및 메뉴 안내
깡통전세방지 상세보기 클릭 시 제휴사 페이지로 이동
