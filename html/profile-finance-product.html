맞춤상품안내

상품 추천 결과를 활용해 상품 안내부터 가입까지 지원하는 Sub Agent예요.

Agent 개요

목적
기존 ML 기반 상품 추천 결과를 활용하여 고객 맞춤형 상품 안내부터 질의응답·가입까지 자연스럽게 연결해요.
추진 배경
단순 추천·노출 중심으로 가입 연계의 한계가 있었고, 추천 이후 금리·우대조건·가입 가능 여부 확인을 위한 별도 탐색과 상담이 필요했어요.
대상 고객
금융 상품(예금·적금·대출·투자·보험) 비교·선택이 필요한 고객
화면 형태
인앱 대화형 채팅 화면 — 상품 카드 + 가입 연결 버튼

역할 및 기능 범위

주요 기능
ML 추천 결과 기반 맞춤형 상품 안내 (1~3순위 제공)
고객 질문에 자연어로 응답 (금리·우대조건·가입 자격 등)
자주 묻는 질문 키워드 사전 제시해 상담 흐름 유도
상품 신청 버튼 클릭 시 금융상품 가입화면 연결
서비스 흐름
노출 — ML 기반 개인화 상품 추천 결과 활용
진입 — AI 금융비서에서 상품 추천 의도 감지 시 자동 전환
안내 — 추천 상품·지식베이스 결합 맞춤형 안내 수행
가입 연결 — 상품 가입 버튼 클릭 시 가입 상세 페이지 이동
