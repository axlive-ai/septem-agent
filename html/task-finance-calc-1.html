UI 컴포넌트 매핑

Flow 1 · 진입
Agent 말풍선(Welcome) — 진입 시 자동 표시
빠른 선택 Chip — 계산 유형 4종, 자유 입력 병행
Flow 2 · 슬롯 수집
슬롯 진행 카드 — 완료·활성·대기 3단계 상태 표시
슬롯별 빠른 선택 Chip + 텍스트 입력창
오류 말풍선 — 범위 초과·비숫자·주제 이탈 시
Flow 3 · 결과
결과 카드 — 월 상환액 강조 + 상세 조건
후속 제안 Chip — 재계산·상품추천·다른계산·종료
